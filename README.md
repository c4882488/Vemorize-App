# [Vemorize-App](https://github.com/c4882488/Vemorize-App)

單字背誦

-------- 

## 介紹

針對孩童製作一款簡易好上手的背誦單字小幫手，對於兒童不需要太複雜的功能，在背誦的時候也可以搭配我們製作的遊戲進行測驗，開發出app與網頁版本。
